正交基线性组合的推导

任意向量可以表示为一组正规正交基 ${v_i}$ 的线性组合，并给出了几个关键公式。让我们一步步推导这些结果。

正交基的性质

首先，我们有一组正规正交基 ${v_i}$，这意味着：

  • $v_i^T v_j = 0$ 当 $i \neq j$ (正交性)
  • $v_i^T v_i = 1$ (单位向量，即正规化)

恒等矩阵的表示

$$I = \sum_i v_i v_i^T$$

推导：
对于正交基，如果我们对任意向量 $x$ 应用上述和式：

$$\left(\sum_i v_i v_i^T\right)x = \sum_i v_i (v_i^T x)$$

由于 $v_i^T x$ 是标量，它表示 $x$ 在 $v_i$ 方向上的投影分量。因此，上式实际上是将 $x$ 分解为各个正交方向的分量，然后再重新组合，结果仍为 $x$。这正是恒等矩阵的作用，所以 $\sum_i v_i v_i^T = I$。

矩阵 $A$ 的谱分解

$$Av_i = \lambda_i v_i \Rightarrow A = \sum_i \lambda_i v_i v_i^T$$

推导：
这里 $v_i$ 是矩阵 $A$ 的特征向量，$\lambda_i$ 是对应的特征值。要证明 $A = \sum_i \lambda_i v_i v_i^T$，我们可以验证对任意特征向量 $v_j$，应用两边得到相同结果：

左边：$Av_j = \lambda_j v_j$

右边：
$$\left(\sum_i \lambda_i v_i v_i^T\right)v_j = \sum_i \lambda_i v_i (v_i^T v_j)$$

由于正交性，当 $i \neq j$ 时，$v_i^T v_j = 0$；当 $i = j$ 时，$v_i^T v_j = 1$。因此：

$$\sum_i \lambda_i v_i (v_i^T v_j) = \lambda_j v_j$$

两边结果相同，因此 $A = \sum_i \lambda_i v_i v_i^T$。这就是矩阵的谱分解表示。

矩阵 $A$ 的逆矩阵

$$A^{-1} = \sum_i \lambda_i^{-1} v_i v_i^T, \quad \text{if } A^{-1} \text{ exists.}$$

推导：
当所有 $\lambda_i \neq 0$ 时，$A$ 是可逆的。我们需要证明 $A \cdot A^{-1} = I$：

$$A \cdot A^{-1} = \left(\sum_i \lambda_i v_i v_i^T\right) \cdot \left(\sum_j \lambda_j^{-1} v_j v_j^T\right)$$

使用矩阵乘法的分配律展开：

$$A \cdot A^{-1} = \sum_i \sum_j \lambda_i \lambda_j^{-1} (v_i v_i^T) (v_j v_j^T)$$

进一步计算内部矩阵乘积：
$$(v_i v_i^T) (v_j v_j^T) = v_i (v_i^T v_j) v_j^T =
\begin{cases}
v_i v_j^T & \text{if } i = j \
0 & \text{if } i \neq j
\end{cases}$$

因此：
$$A \cdot A^{-1} = \sum_i \lambda_i \lambda_i^{-1} v_i v_i^T = \sum_i v_i v_i^T = I$$

这证明了 $A^{-1} = \sum_i \lambda_i^{-1} v_i v_i^T$。

矩阵 $A$ 的伪逆 (Moore-Penrose 逆)

$$A^+ = \sum_{i:\lambda_i\neq 0} \lambda_i^{-1} v_i v_i^T$$

推导：
当矩阵 $A$ 不是满秩（即有些 $\lambda_i = 0$）时，$A$ 不可逆。伪逆 $A^+$ 是逆矩阵概念的推广，仅在非零特征值对应的子空间上计算"逆"。

从计算角度看，伪逆忽略了零特征值对应的特征向量，只在非零特征值对应的子空间上计算逆。故有：

$$A^+ = \sum_{i:\lambda_i\neq 0} \lambda_i^{-1} v_i v_i^T$$

我们可以验证伪逆满足以下性质：

  1. $AA^+A = A$
  2. $A^+AA^+ = A^+$
  3. $(AA^+)^T = AA^+$
  4. $(A^+A)^T = A^+A$

这些性质可以通过将 $A$ 和 $A^+$ 的谱分解表示代入方程并利用正交基的性质来验证。

以上就是正交基线性组合表示矩阵的一系列重要公式的详细推导。

格什戈林圆盘定理（Gershgorin Circle Theorem）详解

格什戈林圆盘定理是矩阵分析中的一个重要结果，它提供了矩阵特征值位置的估计。我将详细解释这个定理及其证明。

定理内容

定理指出：

  1. 设 $A$ 为 $n \times n$ 矩阵，定义 $R_i := \sum_{j \neq i} |a_{ij}|$（即第 $i$ 行除对角元素外所有元素的绝对值之和）
  2. 则 $A$ 的所有特征值都位于复平面上的某些圆盘 $D(a_{ii}, R_i)$ 内

这里，$D(a_{ii}, R_i)$ 表示以对角元素 $a_{ii}$ 为圆心，$R_i$ 为半径的圆盘。

证明解释

证明的关键步骤如下：

  1. 选取特征值和特征向量：

    • 取 $A$ 的一个特征值 $\lambda$ 和对应的特征向量 $v$
    • 设 $i$ 使得 $|v_i|$ 为向量 $v$ 中绝对值最大的分量

  2. 特征值方程：

    • 根据特征值定义 $Av = \lambda v$
    • 这意味着 $\sum_j a_{ij} v_j = \lambda v_i$
    • 将此式改写为 $\sum_{j \neq i} a_{ij} v_j = (\lambda - a_{ii})v_i$

  3. 推导不等式：

    • $|\lambda - a_{ii}| = |\sum_{j \neq i} \frac{a_{ij} v_j}{v_i}|$
    • $\leq \sum_{j \neq i} | \frac{a_{ij} v_j}{v_i}|$
    • $\leq \sum_{j \neq i} |a_{ij}| \cdot |\frac{v_j}{v_i}|$
    • 由于 $|v_i|$ 是最大值，所以 $|\frac{v_j}{v_i}| \leq 1$
    • 因此 $|\lambda - a_{ii}| \leq \sum_{j \neq i} |a_{ij}| = R_i$

  4. 结论：

    • 这表明特征值 $\lambda$ 到对角元素 $a_{ii}$ 的距离不超过 $R_i$
    • 因此 $\lambda$ 必须位于以 $a_{ii}$ 为圆心，$R_i$ 为半径的圆盘内

定理意义

  1. 这个定理提供了矩阵特征值的位置估计，不需要求解特征方程。
  2. 所有特征值都位于 $n$ 个圆盘的并集内。
  3. 如果圆盘互不相交，则每个圆盘恰好包含一个特征值。
  4. 对角占优的矩阵（即 $|a_{ii}| > R_i$）可以确保所有特征值非零。

这个定理在数值分析、矩阵稳定性分析和迭代方法收敛性证明中有广泛应用。
不，行格尔绍戈林圆盘（Row Gershgorin disc）和列格尔绍戈林圆盘（Column Gershgorin disc）不一定等价，但所有特征值都必须位于它们的并集中。让我详细解释一下，并用LaTeX格式化。

定义

  • 行格尔绍戈林圆盘（Row Gershgorin disc）： 对于一个 $n \times n$ 矩阵 $A$，第 $i$ 个行格尔绍戈林圆盘的中心是 $a_{ii}$，半径是该行中非对角线元素的绝对值之和，即 $R_i = \sum_{j \neq i} |a_{ij}|$.
  • 列格尔绍戈林圆盘（Column Gershgorin disc）： 对于一个 $n \times n$ 矩阵 $A$，第 $j$ 个列格尔绍戈林圆盘的中心是 $a_{jj}$，半径是该列中非对角线元素的绝对值之和，即 $C_j = \sum_{i \neq j} |a_{ij}|$.

定理回顾

格尔绍戈林圆盘定理指出：

  • $A$ 的所有特征值位于行圆盘的并集中： $\lambda(A) \subseteq \bigcup_{i=1}^{n} D(a_{ii}, R_i)$
  • $A$ 的所有特征值位于列圆盘的并集中： $\lambda(A) \subseteq \bigcup_{j=1}^{n} D(a_{jj}, C_j)$

行盖尔圆盘和列盖尔圆盘通常不完全等价。 为了更好地估计特征值的位置，通常将行圆盘和列圆盘的并集结合起来考虑。 也就是说：
$\lambda(A) \subseteq \left( \bigcup_{i=1}^{n} D(a_{ii}, R_i) \right) \cap \left( \bigcup_{j=1}^{n} D(a_{jj}, C_j) \right)$
这样的估计会更加精确。

Bauer-Fike定理及其证明

定理陈述

设 $\mu$ 是矩阵 $A+E \in \mathbb{R}^{n\times n}$ 的一个特征值，且存在可逆矩阵 $P$ 使得 $P^{-1}AP=D=\text{diag}(\lambda_1,\lambda_2,\cdots,\lambda_n)$，则有：

$$\min_{\lambda\in\sigma(A)}|\lambda-\mu|\leq |P^{-1}|_p|P|_p|E|_p$$

其中 $|\cdot|_p$ 表示 $p$ 范数，$p=1,2,\infty$。

证明步骤

  1. 首先，我们只需考虑 $\mu\notin\sigma(A)$ 的情况（即 $\mu$ 不是 $A$ 的特征值）。此时 $D-\mu I$ 非奇异。

  2. 设 $x$ 是 $A+E$ 对应于特征值 $\mu$ 的特征向量，则：
    $$(A+E-\mu I)x=0$$

  3. 左乘 $P^{-1}$ 得到：
    $$(D-\mu I)(P^{-1}x)=-(P^{-1}EP)(P^{-1}x)$$

  4. 因此：
    $$P^{-1}x=-(D-\mu I)^{-1}(P^{-1}EP)(P^{-1}x)$$

  5. $P^{-1}x$ 是非零向量。对上式两边取 $p$ 范数：
    $$|(D-\mu I)^{-1}(P^{-1}EP)|_p\geq 1$$

  6. 对角矩阵 $(D-\mu I)^{-1}$ 的范数为：
    $$|(D-\mu I)^{-1}|p=\frac{1}{m},\quad \text{其中} \quad m=\min{\lambda\in\sigma(A)}|\lambda-\mu|$$

  7. 根据范数的性质，有：
    $$|P^{-1}|_p|E|_p|P|_p\geq m$$

这就证明了定理中的不等式 (1.5)。

补充说明

  1. 条件数 $\text{cond}(P)=|P^{-1}||P|$ 是特征值扰动的放大系数。

  2. 由于将 $A$ 对角化的相似变换矩阵 $P$ 不是唯一的，因此定义：
    $$\nu(A)=\inf{\text{cond}(P)|P^{-1}AP=\text{diag}(\lambda_1,\lambda_2,\cdots,\lambda_n)}$$

  3. $\nu(A)$ 称为特征值问题的条件数。当 $\nu(A)$ 较小时，矩阵微小扰动只会带来特征值的微小扰动。

  4. 实际应用中，$\nu(A)$ 难以计算，有时只对一个特定的 $P$ 计算 $\text{cond}(P)$ 来代替 $\nu(A)$。

这个定理的重要意义在于：它给出了矩阵受扰动后，其特征值偏离原特征值的上界估计，这个上界与矩阵的条件数和扰动大小有关。

第 $k$ 个特征值 $\lambda_k$

现在我们看更一般的 $\lambda_k$（第 $k$ 个特征值），它稍微复杂一点，因为它需要在某些限制条件下找最大或最小值。公式有两种等价的表达方式：

第一种表达：$\lambda_k = \min_{x \neq 0, x^\top v_i = 0, \forall i \in {1, \dots, k-1}} \frac{x^\top A x}{x^\top x}$

  • 意思：
    你要找一个方向 $x$，让 $\frac{x^\top A x}{x^\top x}$ 尽量小，但有个条件：$x$ 必须跟前 $k-1$ 个特征向量（$v_1, v_2, \dots, v_{k-1}$）垂直。
  • 通俗解释：
    想象你在 $n$ 维空间里，前 $k-1$ 个特征向量是已经占好的“最佳方向”（对应 $\lambda_1$ 到 $\lambda_{k-1}$）。现在你不能走这些方向，只能找一个新的方向，尽量让 $A$ 的效果最小。这个最小值就是 $\lambda_k$。

第二种表达：$\lambda_k = \max_{U: \dim U = k} \min_{x \in U, x \neq 0} \frac{x^\top A x}{x^\top x}$

  • 意思：
    你先挑一个 $k$ 维的空间 $U$（包含 $k$ 个独立方向），然后在这个空间里找一个方向 $x$，让 $\frac{x^\top A x}{x^\top x}$ 最小。接着，在所有可能的 $k$ 维空间中，挑一个让这个最小值最大的。结果就是 $\lambda_k$。
  • 通俗解释：
    假设你在选一个 $k$ 人团队，每个人代表一个方向。你要保证团队里最差的表现（最小拉伸）尽可能好。$\lambda_k$ 就是你能找到的最优团队的最差表现。

另一种角度：$\lambda_k = \max_{x \neq 0, x^\top v_i = 0, \forall i \in {k+1, \dots, n}} \frac{x^\top A x}{x^\top x}$

  • 意思：
    这次你找一个方向 $x$，让 $\frac{x^\top A x}{x^\top x}$ 尽量大，但 $x$ 必须跟后面的特征向量（$v_{k+1}, \dots, v_n$）垂直。
  • 通俗解释：
    你把后面 $n-k$ 个较小的特征值方向“封锁”掉，只在剩下的空间里找最大的拉伸效果，结果就是 $\lambda_k$。

如果 $A$ 是正定阵，
$$
\frac{x^T A x}{x^T x} = \max_{x\neq 0} \frac{x^T A x}{|x|2^2} = \lambda{\max}(A)
$$
则
$$
\text{cond}_2(A) = |A|_2|A^{-1}|_2
$$

$$
|A|2 = \max{|x|2 \leq 1} \sqrt{x^T A^T A x} = \sqrt{\lambda{\max}(A^T A)}
$$

Cayley-Hamilton 定理证明

定理 (Cayley-Hamilton)

对于任意 $n \times n$ 矩阵 $A$，如果我们定义其特征多项式 $q_A(x) = \det(A - xI)$，则：
$$q_A(A) = 0$$
其中 $0$ 表示 $n \times n$ 的零矩阵。

证明 (sketch)

我们首先考虑 $A$ 是可对角化的情况。

步骤 1: 特征多项式的性质

假设 $A$ 是可对角化的，那么 $q_A(A)$ 也必然是可对角化的。

步骤 2: 特征值与特征向量分析

取 $A$ 的任一特征值 $\lambda \in \mathbb{R}$ 和对应的特征向量 $v$ 满足 $Av = \lambda v$。

步骤 3: 特征多项式作用于特征向量

由步骤 1 的观察可知，$v$ 也是 $q_A(A)$ 的特征向量，且其对应的特征值为：
$$q_A(A)v = q_A(\lambda)v$$

步骤 4: 特征多项式在特征值处的取值

由特征多项式的定义，对于任意特征值 $\lambda$，有：
$$q_A(\lambda) = \det(A - \lambda I) = 0$$

步骤 5: 推导 $q_A(A)$ 的特征值

因此：
$$q_A(A)v = q_A(\lambda)v = 0 \cdot v = 0$$

这表明 $q_A(A)$ 的每个特征向量对应的特征值均为 0。

步骤 6: 结论

由于 $q_A(A)$ 可对角化，且其所有特征值均为 0，因此 $q_A(A)$ 只能是全零矩阵。

对于不可对角化的矩阵，可以通过 Jordan 标准型或极限论证完成证明。

最小二乘法：

求 $\lambda$ 以最小化 $||Ax - \lambda x||_2^2$

$||Ax - \lambda x||_2^2 = ||Ax||_2^2 + \lambda^2 ||x||_2^2 - 2\lambda x^\top Ax$

法线方程：$\lambda = \frac{x^\top Ax}{x^\top x}$, Rayleigh quotient!

定理：

对于实数对称矩阵 $A$，假设近似特征向量 $x$ 满足 $||x||_2 = 1$，并且实数 $\lambda$ 满足 $||Ax - \lambda x||_2 < \epsilon$，则有

$\min_{1 \leq j \leq n} |\lambda_j - \lambda| < \epsilon$

证明：

对于实数对称矩阵，可以选择特征向量 $v_1, v_2, \dots, v_n$ 使得它们是正交且归一化的。 展开 $x = \alpha_1 v_1 + \alpha_2 v_2 + \dots + \alpha_n v_n$

$||Ax - \lambda x||2^2 = \left| \sum{j=1}^n \alpha_j (\lambda_j - \lambda) v_j \right|_2^2$

$= \sum_{j=1}^n \alpha_j^2 (\lambda_j - \lambda)^2 ||v_j||_2^2$

$= \sum_{j=1}^n \alpha_j^2 (\lambda_j - \lambda)^2$

$\geq \min_{1 \leq j \leq n} (\lambda_j - \lambda)^2 \sum_{j=1}^n \alpha_j^2$

其中 $\sum_{j=1}^n \alpha_j^2 = ||x||_2^2 = 1$， 所以 $\epsilon^2 > ||Ax - \lambda x||2^2 \geq \min{1 \leq j \leq n} (\lambda_j - \lambda)^2$

因此 $\epsilon > \min_{1 \leq j \leq n} |\lambda_j - \lambda|$

Euclid范数的酉不变性：

对于向量 $X$ 和酉矩阵 $U$，有：

$|UX|_2 = \sqrt{(UX)^H UX} = \sqrt{X^H U^H UX} = \sqrt{X^H X} = |X|_2$

即：

$|UX|_2 = |X|_2$

无穷范数是最大值的证明

要证明对于复数 ( x_1, x_2, \ldots, x_n )，有

[
\lim_{p \to \infty} \left( |x_1|^p + |x_2|^p + \cdots + |x_n|^p \right)^{1/p} = \max { |x_1|, |x_2|, \ldots, |x_n| },
]

其中 ( |x_i| ) 表示复数 ( x_i ) 的模长。

证明过程如下：

设 ( M = \max { |x_1|, |x_2|, \ldots, |x_n| } )，并假设 ( |x_k| = M ) 对于某个 ( k )。对于任意 ( p > 0 )，有

[
|x_k|^p \leq |x_1|^p + |x_2|^p + \cdots + |x_n|^p \leq n \cdot M^p.
]

取 ( p )-th 根，得到

[
M \leq \left( \sum_{i=1}^n |x_i|^p \right)^{1/p} \leq n^{1/p} \cdot M.
]

当 ( p \to \infty )，( n^{1/p} \to 1 )，因此由夹逼定理，

[
\lim_{p \to \infty} \left( \sum_{i=1}^n |x_i|^p \right)^{1/p} = M.
]

即

[
\lim_{p \to \infty} \left( |x_1|^p + |x_2|^p + \cdots + |x_n|^p \right)^{1/p} = \max { |x_1|, |x_2|, \ldots, |x_n| }.
]

证毕。

Frobenius 范数的详细解释

对于一个 ( m \times n ) 的实（或复）矩阵 ( A )，其 Frobenius 范数定义为：

[
| A |F = \sqrt{ \sum{i=1}^m \sum_{j=1}^n |a_{ij}|^2 }
]

用矩阵的迹表示

[
| A |_F = \sqrt{ \text{tr}(A^T A) } = \sqrt{ \text{tr}(A A^T) }
]

乘法不等式（次可乘性）

如果 ( A ) 是 ( m \times n ) 矩阵，( B ) 是 ( n \times p ) 矩阵，则：
[
| AB |_F \leq | A |_F \cdot | B |_F
]

与谱范数的关系

Frobenius 范数与 谱范数（spectral norm, ( | A |_2 )) 满足：
[
| A |_2 \leq | A |_F \leq \sqrt{r} | A |_2
]
其中 ( r = \text{rank}(A) )。

与奇异值的关系

设 ( \sigma_1, \sigma_2, \ldots, \sigma_r ) 是 ( A ) 的非零奇异值，则：
[
| A |_F = \sqrt{ \sigma_1^2 + \sigma_2^2 + \cdots + \sigma_r^2 }
]

Euclid空间是一个定义了一个二元实函数“内积”（即是向量）的线性空间。

课上内容

给定矩阵 $A \in \mathbb{R}^{m \times n}$，向量 $b \in \mathbb{R}^m$，求 $x \in \mathbb{R}^n$ 使得 $Ax = b$.

  • 假设向量 $b$ 有误差，变成 $b + e$， $x$ 会怎么变化？

  • $x = A^{-1}b \rightarrow x = A^{-1}(b+e)$

  • 该线性方程组的条件数 $\text{cond}(A) = \frac{x \text{的相对误差}}{b \text{的相对误差}}$.

记 $e$ 为 $b$ 的误差，则 $x$ 的误差为 $A^{-1}e$。任取一种向量范数，

$\text{cond}(A) = \max_{e, b \neq 0} \frac{\frac{|A^{-1}e|}{|A^{-1}b|}}{\frac{|e|}{|b|}} = \max_{e, b \neq 0} \frac{|A^{-1}e|}{|e|} \frac{|b|}{|A^{-1}b|} = \max_{e \neq 0} \frac{|A^{-1}e|}{|e|} \max_{x \neq 0} \frac{|Ax|}{|x|}$

  • 矩阵的算子范数 $|A| := \max_{x \neq 0} \frac{|Ax|}{|x|}$

  • 因此条件数 $\text{cond}(A) = |A| \cdot |A^{-1}|$

条件数越大，表示问题越敏感，输入数据的微小扰动可能会导致解的巨大变化。 条件数越接近于1，表示问题越稳定。

严格对角占优矩阵 (Strictly Diagonally Dominant Matrix)

定义：我们称矩阵 $A \in \mathbb{R}^{n \times n}$ 为严格对角占优，如果它满足以下条件：

$\forall i \in {1, 2, …, n}, |a_{ii}| > \sum_{j=1, j \neq i}^{n} |a_{ij}|$

换句话说，对于矩阵的每一行，对角线元素的绝对值必须大于该行所有其他元素绝对值之和。

定理：

如果方阵 $A$ 是严格对角占优的，则 $A$ 是可逆的，而且对于所有向量 $b$ 和初始猜测 $x^{(0)}$，对线性方程组 $Ax = b$ 使用 Jacobi 迭代都会收敛到唯一解。

  1. Jacobi 迭代： Jacobi 迭代是一种求解线性方程组的迭代方法。它通过不断迭代更新解向量 $x$ 的每个分量，直到收敛到一个稳定值。 对于方程 $Ax = b$, Jacobi 迭代的公式可以写成如下形式：

    $x_i^{(k+1)} = \frac{1}{a_{ii}} \left(b_i - \sum_{j=1, j \neq i}^{n} a_{ij} x_j^{(k)} \right)$， 其中 $x_i^{(k)}$ 表示第 $i$ 个分量在第 $k$ 次迭代的值。

  2. 收敛性： 严格对角占优性保证了 Jacobi 迭代的收敛性。这意味着，无论我们选择什么样的初始猜测 $x^{(0)}$，Jacobi 迭代最终都会收敛到线性方程组 $Ax=b$ 的唯一解。

  3. 重要性： 严格对角占优矩阵在数值线性代数中很重要，因为它们保证了某些迭代方法（如 Jacobi 迭代和 Gauss-Seidel 迭代）的收敛性。这在求解大型稀疏线性方程组时非常有用。

ρ(A) 的定义

ρ(A) (读作 “rho of A”) 是矩阵 A 的所有特征值的绝对值的最大值。 更正式地：

$$ ρ(A) = max{|λ_1|, |λ_2|, …, |λ_n|}$$

其中 $$λ_1, λ_2, …, λ_n$$ 是矩阵 A 的所有特征值（包括复数特征值）

命题： $\rho(A) < 1$ 当且仅当 $\lim_{k\to\infty} A^k = 0$ (即 $A^k$ 收敛到零矩阵)。

证明 $\lim_{k\to\infty} A^k = 0 \Rightarrow \rho(A) < 1$：

设 $\lambda$ 是 $A$ 的一个特征值， $v$ 是对应的特征向量。 那么 $Av = \lambda v$。 因此，$A^k v = \lambda^k v$。

假设 $\lim_{k\to\infty} A^k = 0$。 这意味着对于任何向量 $x$， $\lim_{k\to\infty} A^k x = 0$。 特别地，$\lim_{k\to\infty} A^k v = 0$。

所以，$\lim_{k\to\infty} \lambda^k v = 0$。 因为 $v$ 是一个非零向量（特征向量的定义），所以必须有 $\lim_{k\to\infty} \lambda^k = 0$。

这意味着 $|\lambda| < 1$。 因为 $\lambda$ 是 $A$ 的任意特征值，所以 $A$ 的所有特征值的绝对值都小于 1。 因此，$\rho(A) = \max{|\lambda_i|} < 1$。

结论：

$\rho(A) < 1$ 是 $\lim_{k\to\infty} A^k = 0$ 的一个充分必要条件。 也就是说，这两个命题是等价的。 如果一个成立，另一个也一定成立。
当 $A^k \to 0$ (即 $\rho(A) < 1$) 的时候，对任意初始值都收敛的原因如下：

考虑迭代过程 $x_{k+1} = Ax_k + b$。 固定点 $x^$ 满足 $x^ = Ax^* + b$。

方法一：误差分析

定义误差 $e_k = x_k - x^*$。 那么：

$e_{k+1} = x_{k+1} - x^* = (Ax_k + b) - (Ax^* + b) = A(x_k - x^*) = Ae_k$

因此， $e_k = A^k e_0 = A^k (x_0 - x^*)$。

如果 $\rho(A) < 1$，那么 $\lim_{k\to\infty} A^k = 0$。

所以，$\lim_{k\to\infty} e_k = \lim_{k\to\infty} A^k (x_0 - x^*) = 0$。

这意味着 $\lim_{k\to\infty} x_k = x^$，即迭代收敛到固定点，且收敛性与初值$x_0$ 无关。 $|e_k| = |A^k(x_0-x^)| \le |A^k| |x_0 - x^*| \to 0$

方法二：迭代展开

$x_k = Ax_{k-1} + b = A(Ax_{k-2} + b) + b = A^2 x_{k-2} + Ab + b = \dots = A^k x_0 + A^{k-1}b + A^{k-2}b + \dots + Ab + b$

可以写成：
$x_k = A^k x_0 + \sum_{i=0}^{k-1} A^i b = A^k x_0 + (A^{k-1} + A^{k-2} + \dots + A + I)b$

因为 $\rho(A) < 1$，所以 $\lim_{k\to\infty} A^k = 0$。 并且 $\sum_{i=0}^{\infty} A^i = (I - A)^{-1}$ （这是一个矩阵几何级数）。

因此：

$\lim_{k\to\infty} x_k = \lim_{k\to\infty} A^k x_0 + \lim_{k\to\infty} \sum_{i=0}^{k-1} A^i b = 0 + (I - A)^{-1} b = (I - A)^{-1} b$

固定点 $x^$ 满足 $x^ = Ax^* + b$，因此 $x^* = (I - A)^{-1} b$。

所以， $\lim_{k\to\infty} x_k = x^* = (I - A)^{-1} b$。

同样，迭代收敛到固定点，与初值 $x_0$ 无关。

总结

当 $\rho(A) < 1$ 时，误差会随着迭代次数的增加而减小（趋向于0），或者说迭代值会逐渐逼近固定点，与初始值无关。 这两种方法都展示了收敛性，并说明了为什么初值不影响最终的收敛结果。

Jacobi 迭代和 Gauss-Seidel 迭代

1. Jacobi 迭代

  • 原理： Jacobi 迭代是一种求解线性方程组 $Ax = b$ 的迭代方法。它的基本思想是将系数矩阵 $A$ 分解为对角矩阵 $D$，下三角矩阵 $L$ 和上三角矩阵 $U$，即 $A = D - L - U$，然后将方程组转化为 $x = D^{-1}(L + U)x + D^{-1}b$ 的形式，并进行迭代求解。
  • 公式： 将 $A = D - L - U$ 代入 $Ax = b$，得到：
    $(D - L - U)x = b$
    $Dx = (L + U)x + b$
    $x = D^{-1}(L + U)x + D^{-1}b$
    因此，Jacobi 迭代的迭代公式为：
    $x^{(k+1)} = D^{-1}(L + U)x^{(k)} + D^{-1}b$
    或者写成逐个分量的形式：
    $x_i^{(k+1)} = \frac{1}{a_{ii}} \left(b_i - \sum_{j=1, j \neq i}^{n} a_{ij} x_j^{(k)}\right)$, $i = 1, 2, \dots, n$
    其中，$x^{(k)}$ 表示第 $k$ 次迭代的解向量，$a_{ij}$ 是矩阵 $A$ 的元素，$b_i$ 是向量 $b$ 的元素。
  • 收敛性条件： Jacobi 迭代的收敛性取决于迭代矩阵 $B_J = D^{-1}(L + U)$ 的谱半径 $\rho(B_J)$。
    • 如果 $\rho(B_J) < 1$，则 Jacobi 迭代收敛。
    • 充分条件：
      • 矩阵 $A$ 是严格对角占优矩阵（Strictly Diagonally Dominant）：对于所有 $i$，满足 $|a_{ii}| > \sum_{j=1, j \neq i}^{n} |a_{ij}|$。
      • 矩阵 $A$ 是不可约的对角占优矩阵。
  • 算法步骤：
    1. 将矩阵 $A$ 分解为 $D - L - U$。
    2. 计算迭代矩阵 $B_J = D^{-1}(L + U)$ 和向量 $f = D^{-1}b$。
    3. 选择初始向量 $x^{(0)}$。
    4. 迭代计算 $x^{(k+1)} = B_J x^{(k)} + f$，直到满足收敛条件（例如，$|x^{(k+1)} - x^{(k)}| < \epsilon$，其中 $\epsilon$ 是一个很小的正数）。
  • 优点：
    • 每次迭代的计算量较小。
    • 算法简单，容易实现。
  • 缺点：
    • 收敛速度可能较慢。
    • 收敛条件较为苛刻，可能不收敛。
    • 需要存储多个向量，占用内存空间较大。

2. Gauss-Seidel 迭代

  • 原理： Gauss-Seidel 迭代也是一种求解线性方程组 $Ax = b$ 的迭代方法。与 Jacobi 迭代类似，它也将系数矩阵 $A$ 分解为 $A = D - L - U$。不同之处在于，Gauss-Seidel 迭代在计算 $x_i^{(k+1)}$ 时，会立即使用已经计算出的 $x_1^{(k+1)}, x_2^{(k+1)}, \dots, x_{i-1}^{(k+1)}$，而不是像 Jacobi 迭代那样使用上一轮迭代的值。
  • 公式： 将 $A = D - L - U$ 代入 $Ax = b$，得到：
    $(D - L - U)x = b$
    $(D - L)x = Ux + b$
    $x = (D - L)^{-1}Ux + (D - L)^{-1}b$
    因此，Gauss-Seidel 迭代的迭代公式为：
    $x^{(k+1)} = (D - L)^{-1}Ux^{(k)} + (D - L)^{-1}b$
    或者写成逐个分量的形式：
    $x_i^{(k+1)} = \frac{1}{a_{ii}} \left(b_i - \sum_{j=1}^{i-1} a_{ij} x_j^{(k+1)} - \sum_{j=i+1}^{n} a_{ij} x_j^{(k)}\right)$, $i = 1, 2, \dots, n$
  • 收敛性条件： Gauss-Seidel 迭代的收敛性取决于迭代矩阵 $B_{GS} = (D - L)^{-1}U$ 的谱半径 $\rho(B_{GS})$。
    • 如果 $\rho(B_{GS}) < 1$，则 Gauss-Seidel 迭代收敛。
    • 充分条件：
      • 矩阵 $A$ 是严格对角占优矩阵。
      • 矩阵 $A$ 是对称正定矩阵。
  • 算法步骤：
    1. 将矩阵 $A$ 分解为 $D - L - U$。
    2. 选择初始向量 $x^{(0)}$。
    3. 对于 $i = 1, 2, \dots, n$，依次计算 $x_i^{(k+1)} = \frac{1}{a_{ii}} \left(b_i - \sum_{j=1}^{i-1} a_{ij} x_j^{(k+1)} - \sum_{j=i+1}^{n} a_{ij} x_j^{(k)}\right)$。
    4. 重复步骤 3，直到满足收敛条件。
  • 优点：
    • 通常比 Jacobi 迭代收敛速度更快。
    • 只需要存储一个向量，占用内存空间较小。
  • 缺点：
    • 每次迭代的计算量比 Jacobi 迭代略大。
    • 收敛性分析比 Jacobi 迭代更复杂。
    • 不适合并行计算，因为计算依赖于之前的结果。
