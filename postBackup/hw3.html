问题 #1

给定矩阵 $A \in \mathbb{R}^{n \times n}$ 和向量 $b \in \mathbb{R}^n$，最小二乘法可以解决如下问题：

$\underset{x \in \mathbb{R}^n}{\arg\min} |Ax - b|_2$.

现在考虑如下切比雪夫多项式的参数拟合问题：

令 $t(x) = [T_0(x) \ T_1(x) \ T_2(x) \ \dots \ T_n(x)]^T$，其中，对于 $0 \le i \le n$，$T_i(x) = \cos(i\arccos x)$ 是切比雪夫多项式。现在，给定 $n+1$ 个点对 ${(x_i, y_i) \in \mathbb{R}^2 \ | \ 0 \le i \le n}$，请找到 $a \in \mathbb{R}^{n+1}$，使得多项式 $p(x) = a^T t(x)$ 最小化 $\sum_{i=0}^n (p(x_i) - y_i)^2$.

请将该问题翻译为最小二乘法能解决的问题（具体的，应该说明如何设置 $A, b$，以及求解完成之后 $x$ 的含义）.

问题 #2

  1. 假设矩阵 $A \in \mathbb{R}^{n \times m} (n \le m)$ 的行是线性无关的，请证明 $AA^T$ 是可逆的，并找出矩阵 $A^+ \in \mathbb{R}^{m \times n}$ 使得 $AA^+ = I_n$. (HINT: $A^+$ 的表达式里面可以出现 $(AA^T)^{-1}$)

  2. 对于有无穷组解的线性方程组 $Ax = b$，请证明 $x = A^+b$ 是这些解里面 2-范数最小的. (HINT: 可以尝试从几何直观中思考，长度什么时候最短)

  3. 对于向量 $x \in \mathbb{R}^n$ 和整数 $p > 0$, $x$ 的 $p$-范数定义为 $||x||p \triangleq (\sum{i=1}^n |x_i|^p)^{\frac{1}{p}}$. 请证明 $\lim_{p \to \infty} ||x||p = ||x||\infty = \max_{1 \le i \le n} |x_i|$.

  4. 对于向量 $x \in \mathbb{R}^n$, 请证明 $||x||_\infty \le ||x||2 \le \sqrt{n} ||x||\infty$.
    问题 #3

在这个题目中，我们将考虑快速傅立叶变换 (FFT).

回顾: 对于多项式 $p(x) = a_0 + a_1 x + a_2 x^2 + \dots + a_n x^n$，快速傅立叶变换是找到 $\omega \in \mathbb{C}$ 为 $n+1$ 次单位根，计算多项式在下列点,

$ \forall 0 \le j \le n, \quad p(\omega^j) = \sum_{i=0}^n a_i \omega^{ij} $,

快速傅立叶逆变换的目的是找到系数 $a_i$，计算多项式在下列点,

$ \forall 0 \le i \le n, \quad a_i = \frac{1}{n+1} \sum_{j=0}^n p(\omega^j) \omega^{-ij} $,

其中，逆变换系数 $\sum_{j=0}^n \omega^{ij}$ 在当 $i \ne 0$ 需要满足 $\omega = 0$.

快速傅立叶变换 (FFT) 在实践中的阶数需要处理整数，这会带来一些困难。为了处理这个问题，在这个问题中，我们将探索如何在特定运算的意义下进行快速傅立叶变换。我们将只考虑 $\pmod{5}$.

  1. 首先我们来验证一些简单的事实:

    (a) 对于 $z \in {1, 2, 3, 4}$，计算 $z^4 \pmod{5}$.

    (b) 计算，对于 $\omega = 3$,
    $ {\omega^0, \omega^1, \omega^2, \omega^3 } \equiv ? \pmod{5} $
    $ {\omega^0, \omega^2, \omega^4, \omega^6 } \equiv ? \pmod{5} $
    $ {\omega^0, \omega^3, \omega^6, \omega^9 } \equiv ? \pmod{5} $

    (c) 对于 $\omega = 3$，计算 $1 + \omega + \omega^2 + \omega^3 \pmod{5}$.

(提示: $\omega = 3$ 在 $\pmod{5}$ 的意义下和 $\omega = i$ 有相似的性质. 利用这些发现，我们可以在 $(\pmod{5})$ 意义下计算快速傅立叶变换.

  1. 令 $\omega = 3$，用 FFT 求出多项式 $3x + 2x^2$ 在 ${1, 3, 4, 2}$ 的值 $(\pmod{5})$，写出递归算法运行的大致过程.

  2. 某一个多项式在 ${1, 3, 4, 2}$ 上求出的值分别是 ${0, 0, 1, 4} \pmod{5}$，请通过 FFT 的逆变换求出这个多项式，写出递归算法运行的大致过程.

提示: FFT 的逆向 FFT 过程几乎一样，但是需要使用 $\omega^{-1} \equiv 2 \pmod{5}$，而不是 $\omega = 3$. 并且我们还需要在执行完算法之后乘上 $4^{-1} \pmod{5}$ 进行归一化.

  1. 利用 $(\pmod{5})$ 意义下的 FFT 来计算 $3x + 2x^2 + x^3$ 的乘积 $(\pmod{5})$.

问题 #4

对任意多项式 $f, g$，考虑内积

$\langle f, g \rangle := \int_{-1}^{1} f(x)g(x) \frac{dx}{\sqrt{1-x^2}}$,

利用该内积，可以定义范数 $||f|| := \sqrt{\langle f, f \rangle}$.

现在令 $T_n$ 表示 $n$ 阶切比雪夫多项式. 对于 $d > 1$，请找出 $a_0, a_1, \dots, a_{d-1}$ 使得 $|| T_d - (a_0 T_0 + a_1 T_1 + \dots + a_{d-1} T_{d-1}) ||$ 最小.

提示：切比雪夫多项式关于这个内积正交.
